How it works

Customers can order as guests or registered users. Servers can take orders by phone. Your order always stays linked by phone number.

Guest: Enter phone → choose items → place order.
Register later: Same phone → we auto-link old orders.
Server flow: Assign table → take draft order → review → send to kitchen.
Tip: Use the seeded accounts and password password to test roles.
